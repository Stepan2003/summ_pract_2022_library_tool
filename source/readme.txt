(«Импорт данных», «Сохранить таблицу», «Справка»)> Ниже находятся кнопки «Таблица выдачи и получения книг», «Книги», «Ученики/филиалы». 	Каждая из кнопок отображает одну из таблиц из Базы Данных с соответствующим содержанием.    -----> ДОБАВЛЕНИЕ НОВЫХ ЗАПИСЕЙ <----- (ВЫДАЧА КНИГ)> Чтобы добавить новую запись в отображённую таблицу нужно:	1) Нажать на кнопку «Добавить запись».	2) Заполнить все поля.	3) Нажать кнопку «Сохранить».> ВЫДАЧА КНИГ осуществляется через добавление записи в таблице выдачи и получения книг.        -----> РЕДАКТИРОВАНИЕ <----- 	(ВОЗВРАТ КНИГ)> Чтобы открыть ОКНО РЕДАКТИРОВАНИЯ строки, выполните двойной щелчок Левой Кнопкой Мыши (ЛКМ) на целевой строке.> Некоторые поля недоступны для редактирования в связи со структурой БД.> ВОЗВРАТ КНИГ осуществляется через окно редактирования в таблице выдачи и получения книг.	1) Выполните двойной щелчок ЛКМ по строке с несданной книгой.	2) Нажмите кнопку «Вернуть книги».	3) Программа автоматически выставит текущую дату в соответствующее поле. Книга будет считаться возвращенной в библиотеку.    -----> УДАЛЕНИЕ СТРОК ИЗ ТАБЛИЦ <----- > Удаление ОДНОЙ записи:	1) Нажмите ЛКМ по соответствующей строке. Она должна выделиться.	2) Нажмите на клавиатуре клавишу Delete.	3) Следуйте подсказкам диалоговых окон.> Удаление НЕСКОЛЬКИХ записей:	1) Если нужно, отобразите в таблице определенные данные, используя функцию фильтрации (см.далее).	2) Нажмите на кнопку «Удалить отображенные» в правом верхнем углу окна.	3) Следуйте подсказкам диалоговых окон.    -----> СОРТИРОВКА <----- > Для СОРТИРОВКИ строки таблицы по одному из столбцов, нажмите на него Правой Кнопкой Мыши (ПКМ). > Можно отсортировать в обратном порядке, повторно нажав по столбцу ПКМ.        -----> ФИЛЬТРАЦИЯ <----- > Чтобы ФИЛЬТРОВАТЬ значения таблицы, введите значения в поля фильтра. > При переключении между таблицами значения в полях фильтра сохраняются.> Чтобы сбросить фильтры нажмите кнопку «Очистить поля» или удалите все символы из полей вручную.> Переключатель «Искать с первого символа» изменяет алгоритм фильтрации.     -----> СОХРАНЕНИЕ СТРОК ИЗ ТАБЛИЦЫ <----- (ВЫПИСКА, ЧИТАТЕЛЬСКИЙ БИЛЕТ)> Чтобы сохранить строки, отображенные в текущей таблице, 	1) Нажмите кнопку «Сохранить таблицу» в верхней панели или в правом верхем углу окна или комбинацию клавиш Ctrl+S.	2) Выберите папку сохранения и, если нужно, задайте имя файлу.	3) Нажмите кнопку «Сохранить».	Файл будет сохранён в формате CSV, который можно открыть и редактировать с помощью программы Excel или текстового редактора (кодировка «windows-1251», поддерживает кириллицу)    -----> ИМПОРТ ДАННЫХ ИЗ ФАЙЛА <-----  > Программа поддерживает импорт данных сохранённых в формате CSV.
